About SXC Code Hiking

The SXC CODE HIKING is the event organized by St. Xavier’s College, Maitighar. This event is a platform for aspiring tech enthusiasts to start a global initiative to connect with the surrounding around us. As the world's first hackathon held within a forest, we inspire global institutions to reimagine the possibilities of tech-driven events. Join us as we embark on a journey to blend coding challenges with the serenity of nature, fostering learning, collaboration, and unforgettable experiences..
